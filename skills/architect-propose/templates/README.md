# Architect Propose Templates

This directory contains two different kinds of reference material:

- The numbered root template files are the source fragments used by the plan
  creation and document allocation scripts.
- `example-plan/` is one complete sealed plan package written in English. It
  mirrors the real package contract exactly, except that it is stored here as a
  study reference instead of under `.architect/`.

Use the single-file templates only through the repository Python scripts when
generating a live package. Do not hand-create the initial package tree or
initial package files. Inspect `example-plan/` when you need to understand the
complete directory shape, cross-document references, initialized state, and
final sealed form of a plan package.
